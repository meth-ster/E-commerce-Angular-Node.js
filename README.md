# E-commerce-Angular-Node.js
A full-stack e-commerce application built with Angular and Node.js, using TypeScript as the primary language.

## What it does
This application allows users to browse products, add them to cart, and checkout securely. It also includes administrative features for managing products, orders, and users.

## Installation and Running
To get started, clone the repository and run the following commands:
```bash
npm install
npm run build
npm run start
```
This will install dependencies, build the application, and start the server. You can then access the application at `http://localhost:3000`.

## Example Usage
To test the application, simply browse to `http://localhost:3000` and navigate through the product catalog. You can add products to your cart and proceed to checkout to see the application in action.
